模組化影片/音訊播放器

簡單易用，適合個人網站嵌入。

輕鬆嵌入到個人網站

只需從 cdn.jsdelivr.net 引入核心源碼

https://cdn.jsdelivr.net/npm/pdplayerkit@1.0.0/dist/PDPlayerKit.min.js

使用 JavaScript 代碼添加播放器

                
const dom = new PDPlayer({
    // 選擇播放器風格，例如 "minimal" 或 "classic"
    type: "[風格選擇]",

    // 媒體來源設定：以下個選項中僅能選擇一項
    video: "[影片位置]",
    // audio: "[音訊位置]",
    // vimeo: "[Vimeo ID]",
    // youtube: "[YouTube ID]"

    // 設定顯示的控制面板選項
    panel: [
        "play", "progress", "time", 
        "volume", "volumeMini", "rate", "full"
    ],      

    // 播放設置           
    volume: 100,
    mute: false,

    // 事件監聽
    event: {
        ready: function () {
            console.log("播放器準備完成");
        },
        playing: function () {
            console.log("播放中");
        },
        pause: function () {
            console.log("暫停");
        },
        end: function () {
            console.log("播放結束");
        }
    }
});

(...).appendChild(dom.body);

多種不同風格的面板做選擇

提供多種預設面板風格，方便快速使用
風格與按鈕皆可自訂

可拖曳變換位置

直接提供影片 ID 即可播放
